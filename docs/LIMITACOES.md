# Limitações reais

1. O sistema não prevê todos os desligamentos.
2. Corte instantâneo de energia pode não gerar alerta.
3. Kernel-Power 41 não aponta causa sozinho.
4. Sensores de temperatura podem não estar disponíveis pelo Windows.
5. Diagnóstico físico ainda pode ser necessário.
6. Logs contínuos são salvos antes, mas relatório completo normalmente é salvo depois do boot.
7. Relatório parcial só existe se o Windows detectar algum sinal antes da falha.
8. Análise profunda de dump exige WinDbg.
9. O projeto não substitui teste de fonte, placa-mãe, RAM, SSD, cabo ou tomada.
10. O diagnóstico separa evidências, suspeitas e hipóteses; não deve ser tratado como certeza absoluta.
