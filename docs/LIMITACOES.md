# Limitacoes reais

1. O sistema nao preve todos os desligamentos.
2. Corte instantaneo de energia pode nao gerar alerta.
3. Kernel-Power 41 nao aponta causa sozinho.
4. Sensores de temperatura podem nao estar disponiveis pelo Windows.
5. Diagnostico fisico ainda pode ser necessario.
6. Logs continuos sao salvos antes, mas relatorio completo normalmente e salvo depois do boot.
7. Relatorio parcial so existe se o Windows detectar algum sinal antes da falha.
8. Analise profunda de dump exige WinDbg.
9. O projeto nao substitui teste de fonte, placa-mae, RAM, SSD, cabo ou tomada.
10. O diagnostico separa evidencias, suspeitas e hipoteses; nao deve ser tratado como certeza absoluta.
